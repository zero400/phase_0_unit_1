Apply CSS to position elements on an HTML page

For me, this is far simpler with an external linking style sheet but lets do it internally in the style tags.

One important concept to brush up on is the [Box:Model]. It means that every object placed on a page has the shape of a box. Each box has a width and height. You can color rounded corners and have paddings and change text color but all those styles will apply to inside each box.

Compare different HTML tags to one another
